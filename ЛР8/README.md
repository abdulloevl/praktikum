Задача 1. 
Filtering + Mapping (list comprehension)
nums = [1,2,3,4,5,6]
squares = [x*x for x in nums if x % 2 == 0] print(squares)
Цель программы — выполнить фильтрацию и преобразование данных с использованием механизма list comprehension.
Программа принимает список чисел и:
1.	фильтрует только чётные числа
2.	преобразует их, возводя каждое число в квадрат
Использование list comprehension позволяет объединить фильтрацию и преобразование в одной компактной и эффективной конструкции, улучшая читаемость и производительность кода.

Учебная нагрузка (посещаемость)

1 — студент был на занятии; 
0 — отсутствовал; 
список = посещаемость по дням. Генерация данных 
attendance = [random.choice([0, 1]) for _ in range(30)]

1.	процент посещаемости; 
2.	максимальная серия 1. 

 Генерация данных:
1.	Создаём список длиной 30, где каждый элемент — случайное число 0 или 1.
2.	Пример: [1, 0, 1, 1, 0, 1, ...]
  Обработка данных:
1.	Процент посещаемости = (количество дней с посещением) / (общее количество дней) × 100
2.	Максимальная серия посещений = наибольшее количество подряд идущих 1 в списке
  Вывод результата:
1.	Печать посещаемости по дням
