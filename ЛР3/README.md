Задача 2.  Нужно найти произведение элементов списка.
Для этого создают специальную переменную-аккумулятор p, в которую по очереди перемножаются все числа из списка.
В примере используется список [2, 3, 4]. Программа должна перемножить эти числа и вывести результат.Бизнес-логика решения:
Создаём переменную p = 1, которая будет служить аккумулятором произведения.
Берём список чисел, например [2, 3, 4].
Запускаем цикл for, который будет проходить по каждому числу списка.
На каждом шаге умножаем аккумулятор p на текущее число x.
После окончания цикла p содержит итоговое произведение всех элементов списка.
Выводим результат на экран.
