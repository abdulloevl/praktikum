Задача 1. hекурсивно определить максимальную глубину вложенности списка. 
Пример: 
[1, [2, [3, [4]]]] 
Результат: 4 
Важно: пустой список тоже учитывать. 

Задача 2Цель программы — определить минимальную сумму последовательных элементов длины k в списке положительных чисел.
Программа принимает набор положительных чисел и значение k, которое определяет длину анализируемого подмассива.
Используется алгоритм Sliding Window (скользящее окно), позволяющий эффективно вычислять сумму без повторных пересчётов.
На первом шаге вычисляется сумма первых k элементов.
Затем окно последовательно сдвигается вправо на один элемент:
1.	новый элемент добавляется к текущей сумме
2.	элемент, вышедший из окна, вычитается
На каждом шаге текущая сумма сравнивается с минимальной.
В конце работы программа возвращает минимальную найденную сумму.
